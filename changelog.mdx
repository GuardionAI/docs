> New updates and improvements

<Update label="May 2025">
  ## Enhanced Security Monitoring & Feedback

  <Frame>
<img src="/images/logs-view.png" alt="Guardion investigation dashboard showing filtered logs and threat details" />
</Frame>

  ### Logs & Investigation
  - **Improved Logs Interface**: Redesigned logs listing for easier tracking of details and historical interactions
  - **Logs API Performance**: Enhanced retrieval speed with expanded search capabilities and filtering options

  ### Feedback System
  - **Bulk Feedback**: Added support for submitting multiple feedback items simultaneously
  - **Integrated Feedback Loop**: New feedback system directly connects with Guard API evaluation for immediate application
  - **Policy Dataset**: Created consolidated dataset of historical feedback for robust review and policy retraining

  ### Dashboard Improvements
  - **Top Threats Visibility**: Dashboard now displays top 10 threats detected, prioritized by lowest confidence scores to highlight areas requiring attention

</Update>
