# Guardion-1-8B Model Card

Guardion-1-8B is a multilingual AI safety judge optimized for grounding verification, hallucination detection, and custom criteria evaluation in RAG and agentic workflows.

> This is the model page for the **Grounding** and **Custom** detectors. See the detector overviews in [Grounding](/grounding) and [Custom](/custom).

---

## Model Card

### Overview

- **Architecture:** Decoder-only transformer
- **Parameters:** 8 billion
- **License:** Apache 2.0
- **Developed by:** Guardion AI

### Key Features

- **Multilingual support** — trained on diverse language data for cross-lingual safety evaluation
- **Hybrid thinking mode** — operates in thinking or non-thinking mode; thinking mode produces detailed reasoning traces via `<think>...</think>` and `<score>...</score>` tags
- **Custom criteria** — bring your own judging criteria for domain-specific policy enforcement without additional training
- **LoRA-compatible** — suitable for LoRA adapters, enabling efficient task-specific fine-tuning on custom safety taxonomies
- **Pre-baked safety criteria** — ships with built-in support for harm detection, RAG hallucination, function calling validation, and more

### Pre-baked Criteria

The model is trained to judge content against these built-in criteria:

**Harm categories:**
- Social Bias, Jailbreaking, Violence, Profanity, Sexual Content, Unethical Behavior, Harm Engagement, Evasiveness

**RAG hallucination:**
- Context Relevance, Groundedness, Answer Relevance

**Agentic workflows:**
- Function Calling Hallucination

## How to Use Guardion-1-8B

Combine Guardion-1-8B with a guard policy, then evaluate with that policy.

### Example: Grounding check

```js
// 1) Create a Grounding policy powered by Guardion-1-8B
await fetch("https://api.guardion.ai/v1/policies", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
  },
  body: JSON.stringify({
    id: "grounding-check",
    definition: "Detect hallucinations and ungrounded claims",
    target: "assistant",
    detector: {
      model: "guardion-1-8b",
      expected: "block",
      threshold: 0.9
    }
  })
});

// 2) Evaluate using that policy
const response = await fetch("https://api.guardion.ai/v1/guard", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
  },
  body: JSON.stringify({
    messages: [
      { role: "user", content: "What is the return policy?" },
      { role: "assistant", content: "Returns are accepted within 30 days." }
    ],
    context: [
      { text: "Our return policy allows returns within 14 days of purchase." }
    ],
    override_enabled_policies: ["grounding-check"]
  })
});

const result = await response.json();

if (result.flagged) {
  console.log("Hallucination detected:", result.reason);
} else {
  console.log("Response is grounded");
}
```
